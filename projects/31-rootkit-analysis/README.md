# Project 31: Study rootkits and how to remove them

## Overview

Guide to understanding and detecting rootkits.

## Types

| Type | Level | Difficulty |
|------|-------|------------|
| User-mode | App level | Medium |
| Kernel-mode | OS level | Hard |
| Bootkit | MBR | Very Hard |
| Firmware | BIOS/UEFI | Extreme |

## Detection Tools

- rkhunter - Linux rootkit detector
- chkrootkit - Linux rootkit scanner
- GMER - Windows rootkit detection
- Volatility - Memory forensics

## Key Principle

**Rootkits persist deep in system. Often only solution is rebuild from clean source.**
