# Project 30: Create a secure cryptocurrency wallet

## Overview

Guide to securing cryptocurrency wallets and assets.

## Wallet Types

| Type | Security | Use Case |
|------|----------|----------|
| Hardware | Max | Large holdings |
| Software | Medium | Daily use |
| Paper | High | Cold storage |

## Key Security Practices

1. **Hardware wallet** for significant holdings
2. **Seed phrase** - never share, metal backup
3. **Multisig** for large amounts
4. **Verify addresses** before sending

## Common Scams

- Phishing sites
- Clipboard swap malware
- SIM swapping
- Fake "support"
