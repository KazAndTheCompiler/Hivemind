# Project 41: Run a red team engagement

## Overview

Structured red team operation framework and methodology.

## Red Team vs Penetration Testing

| Aspect | Pentest | Red Team |
|--------|---------|----------|
| Scope | Specific systems | Entire organization |
| Goal | Find vulnerabilities | Test detection/response |
| Duration | Days-weeks | Weeks-months |
| Approach | Technical | Full adversary simulation |

## Engagement Phases

1. **Recon** - OSINT, footprint mapping
2. **Initial Access** - Phishing, exploits
3. **Persistence** - Implants, scheduled tasks
4. **Lateral Movement** - Pass-the-hash, PTH
5. **Privilege Escalation** - Kernel exploits, token manipulation
6. **Exfiltration** - Data staging, encrypt, extract

## Rules of Engagement

- Written scope
- Emergency contacts
- Kill switches
- Data handling procedures
