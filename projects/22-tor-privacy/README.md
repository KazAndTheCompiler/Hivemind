# Project 22: Use anonymous tools like Tor for privacy

## Overview

Guide to using Tor and understanding privacy tools.

## Usage

```bash
cd 22-tor-privacy
./tor_guide.sh
```

## Key Concepts

| Concept | Description |
|---------|-------------|
| Onion routing | Layered encryption |
| Exit node | Final relay before destination |
| Tor circuit | Complete path through network |

## Installation

```bash
# Debian/Ubuntu
sudo apt install tor

# Run Tor Browser
./tor-browser_en-US/Browser/start-tor-browser
```

## Legitimate Uses

- Journalist-source communication
- Bypassing censorship
- Privacy protection
- Security research
