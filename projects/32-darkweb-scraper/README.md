# Project 32: Build a dark web scraper for research

## Overview

Framework for ethical dark web research.

## ⚠️ Legal Warning

## Setup

```bash
# Install Tor
sudo apt install tor

# Start Tor service
sudo systemctl start tor

# Run research
python3 scraper.py
```
